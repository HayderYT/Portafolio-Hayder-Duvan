Contenido del proyecto
Este proyecto incluye una página web tipo portafolio profesional con:

Página principal con nombre completo, fotografía, perfil, objetivos y datos de contacto.
Sección académica con formación, competencias técnicas, lenguajes y certificación.
Sección de proyectos con dos proyectos, descripciones, tecnologías, capturas e hipervínculos.
Integración de voz por sección: cada botón reproduce un resumen de lo que se ve en esa sección.
Archivos de audio MP3 incluidos como respaldo en assets/audio/.
JavaScript para navegación, animaciones, progreso de lectura, menú responsive y control de voz.
Documento corto de explicación en docs/documento_portafolio_hayder.pdf.
Capturas de funcionamiento en capturas/.

Cómo ejecutar
Abre la carpeta portfolio-hayder-duvan.
Haz doble clic en index.html para abrirlo en el navegador.
Para probar la voz, presiona los botones Escuchar resumen o usa el panel flotante Voz del portafolio.
Audio
La página usa primero la voz natural disponible en el navegador mediante Web Speech API. Si el navegador no tiene voz en español o bloquea la reproducción, usa como respaldo los archivos MP3 ubicados en assets/audio/.
