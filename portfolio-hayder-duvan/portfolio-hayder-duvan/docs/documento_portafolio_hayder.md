# Documento corto - Portafolio Web Personal

**Estudiante:** Hayder Duvan Carreño Ramos  
**Programa:** Ingeniería de Software  
**Actividad:** Creación de un Portafolio Web Personal Interactivo con Integración de Audio

## Objetivo del portafolio

El objetivo del portafolio es presentar de forma organizada el perfil académico y profesional del estudiante, integrando información personal, formación académica, competencias técnicas, proyectos desarrollados y un sistema de audio que resume cada sección para mejorar la experiencia de usuario.

## Tecnologías utilizadas

- HTML5 para la estructura semántica de la página.
- CSS3 para el diseño visual, responsive, tarjetas, animaciones y adaptación a diferentes dispositivos.
- JavaScript para el menú móvil, animaciones de aparición, barra de progreso, control de voz por sección y reproducción de audios.
- Web Speech API para usar voces naturales del navegador cuando estén disponibles.
- Archivos MP3 como respaldo para cumplir con el entregable de audio.

## Proyectos incluidos

1. **Plataforma Caddies - Club de Tenis Colombia:** sistema web para apoyar la gestión de canchas, empleados, turnos, pagos y reportes.
2. **Registro de Estudiantes:** aplicación web para registrar, editar, eliminar y buscar estudiantes con una interfaz visual y ordenada.

## Dificultades encontradas

Una de las dificultades principales fue organizar toda la información en una sola página sin que se viera saturada. Para solucionarlo se dividió el contenido en secciones claras y se aplicaron tarjetas visuales. Otra dificultad fue la integración de audio con una voz más natural; por eso se implementó el uso de la voz del navegador como opción principal y archivos MP3 como respaldo.

## Conclusiones

El portafolio permite evidenciar competencias en desarrollo web, organización de archivos, diseño responsive e integración multimedia. Además, la función de audio por sección mejora la accesibilidad y hace que el sitio sea más interactivo. Este trabajo fortalece habilidades importantes para la formación en Ingeniería de Software, como documentación, estructura de código y experiencia de usuario.
